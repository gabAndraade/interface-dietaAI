# 🥗 Dieta.AI

Aplicativo mobile desenvolvido para geração de dietas personalizadas utilizando Inteligência Artificial.

O projeto analisa os objetivos do usuário e gera sugestões de refeições de forma automática, oferecendo uma experiência simples, moderna e intuitiva.

# Arquitetura do Projeto

O aplicativo foi desenvolvido utilizando React Native com TypeScript, focando em componentização, organização de código e integração com API REST.

---

# Integração com Backend

O front-end se comunica com uma API desenvolvida em Node.js + Fastify responsável pela geração das dietas e processamento das requisições.

---

# Como executar o projeto

## Clone o repositório

```bash
git clone <url-do-repositorio>
```

## Instale as dependências

```bash
npm install
```

## Execute a aplicação

```bash
npx expo start
```
